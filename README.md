# momentum-readme

Momentum is a lightweight weekly goal tracker built by Sherpa-4-Good as a real world learning product.

This repository documents the project. The live demo and full access are **not publicly available**.

## What we built
Momentum allows users to:
- Set weekly goals
- Mark goals as complete
- See real time progress
- Keep data stored locally

The focus is simplicity, clarity, and shipping something usable.

## Why this exists
Momentum was created to practice real product development:
- Clean frontend architecture
- Thoughtful UI decisions
- Version control and collaboration
- Building an MVP with clear scope

## Tech overview
- HTML
- CSS
- JavaScript
- Browser LocalStorage

No backend. No accounts. No external services.

## Access
Momentum is currently in limited access.

If you’re interested in seeing a demo or learning how it was built, you’ll need to request access.

Active development and iteration.
